EconGym: A Scalable AI Testbed with Economic Benchmarks

From pensions to market design, EconGym supports 25+ policy scenarios

Code Paper

Overview

EconGym is a modular, multi-agent economic simulation testbed designed to support AI-driven policy evaluation and algorithmic benchmarking. It provides a unified interface for simulating fiscal, monetary, market, and individual-level dynamics using diverse agent types, including rule-based, reinforcement learning, and LLM-based agents.

It supports over 25 benchmark tasks, each addressing a core research questionâ€”from pension sustainability to algorithmic collusion. Every task is defined by a YAML configuration, which specifies agent roles, economic dynamics, and evaluation metrics.

How to Use

To run a benchmark simulation, use the following command:

python main.py --problem_scene "delayed_retirement"

Each benchmark scenario is defined in a YAML file under the cfg/ directory. Users can also customize economic roles and agent strategies via YAML configuration.
